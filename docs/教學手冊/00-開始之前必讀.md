# 📖 龜馬山 goLine 平台 - 完整開發教學手冊

> 本教學手冊專為新手設計，像樂高操作手冊一樣，一步一步帶您完成整個系統的開發。

## 🎯 這本手冊的目標

這本手冊會教您從零開始建立「龜馬山 goLine 平台」，包括：
- LINE 平台的完整配置（LINE Login、LIFF、Messaging API）
- Firebase 四個專案的設定
- 前端和後端程式碼開發
- 部署到正式環境
- 維護和更新系統

## 📚 手冊結構

本手冊分為以下章節，**請按順序閱讀**：

### 第一部分：準備工作
- **01-系統架構說明.md** - 了解整個系統的運作方式
- **02-需要準備的帳號和工具.md** - 註冊所有必要的帳號

### 第二部分：LINE 平台配置
- **03-LINE-Login-設定完整步驟.md** - 設定 LINE Login（網頁登入）
- **04-LIFF-設定完整步驟.md** - 設定 LIFF（LINE 內建瀏覽器）
- **05-LINE-Messaging-API-設定.md** - 設定 LINE 官方帳號機器人

### 第三部分：Firebase 配置
- **06-Firebase-專案建立.md** - 建立四個 Firebase 專案
- **07-Firebase-Functions-部署.md** - 部署後端程式

### 第四部分：程式開發
- **08-專案結構說明.md** - 了解程式碼的組織方式
- **09-開發第一個模組.md** - 跟著範例開發新功能
- **10-權限管理系統.md** - 設定使用者權限

### 第五部分：部署和維護
- **11-部署到-Vercel.md** - 發布到正式環境
- **12-日常維護和更新.md** - 如何修改和更新系統
- **13-常見問題解決.md** - 遇到問題怎麼辦

## 🎓 學習建議

### 如果您是完全新手
1. **不要跳過任何章節**，每一章都很重要
2. **照著步驟做**，不要自己亂改
3. **遇到不懂的地方**，先看「常見問題解決」章節
4. **保存所有的 ID 和密鑰**，寫在安全的地方

### 如果您有一些基礎
1. 可以快速瀏覽第一部分（了解架構）
2. 重點看 LINE 和 Firebase 的配置步驟
3. 開發部分可以參考範例自己發揮

### 預計學習時間
- **完整學習**：約 2-3 天（包含所有配置和測試）
- **快速建置**：約 1 天（已有帳號，照著步驟做）

## ⚠️ 重要提醒

### 安全性
- **絕對不要**把 API Key、Secret 放在程式碼裡
- **絕對不要**把密鑰上傳到 GitHub
- **一定要用** Replit Secrets 或環境變數管理密鑰

### 費用
- Firebase Blaze 方案：依用量計費（小型專案約 $0-5/月）
- Vercel Pro 方案：$20/月（已升級）
- LINE 平台：免費

### 備份
- **定期備份**程式碼到 GitHub
- **定期匯出**重要資料（簽到記錄等）
- **記錄所有**配置步驟和 ID
