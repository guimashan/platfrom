## ⚠️ 重要提醒

### 安全性

- **絕對不要**把 API Key、Secret 放在程式碼裡
- **絕對不要**把密鑰上傳到 GitHub
- **一定要用** Replit Secrets 或環境變數

### 備份

- **定期備份**程式碼到 GitHub
- **定期匯出**重要資料
- **記錄所有**配置步驟和 ID

### 測試

- **先在測試環境**測試新功能
- **確認沒問題**再部署到正式環境
- **保留舊版本**以便回滾
